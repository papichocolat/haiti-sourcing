# Haiti Sourcing

Haitian grocery delivery website — delivering authentic groceries to Port-au-Prince.

## About

Haiti Sourcing connects customers in the US, Canada, and abroad with their families in Port-au-Prince. We source, pack, and deliver quality Haitian groceries directly to your family's door.

## Features

- Browse 42+ authentic Haitian grocery products
- Filter by category (Staples, Dairy, Meat, Condiments, Drinks, Packages)
- Order directly via WhatsApp
- Fully mobile-friendly
